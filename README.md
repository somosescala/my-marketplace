# Somos Escala — Marketplace de Plugins

Plugins de marketing digital para Claude Code.

## Instalar el marketplace

```
/plugin marketplace add somosescala/my-marketplace
```

## Plugins disponibles

| Plugin | Descripción |
|--------|-------------|
| `meta-campaign-analyzer` | Analiza campañas de Facebook e Instagram con las 3 Q's |

## Instalar un plugin

```
/plugin install meta-campaign-analyzer@somosescala
```
